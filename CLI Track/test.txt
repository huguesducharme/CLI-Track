Приложение должно работать из командной строки, принимать пользовательские действия и входные данные в качестве аргументов и хранить задачи в файле JSON. Пользователь должен иметь возможность:

 ++++++++ Добавляйте  
 ++++++++ Удаляйте задачи
 ++++++++ Просмотреть все задачи 
 ++++++++ Перечислите все задачи, которые выполняются
 ++++++++ Перечислите все задачи, которые не выполнены
 ++++++++ Перечислите все задачи, которые находятся в процессе



Свойства Задач
Каждая задача должна обладать следующими свойствами:

id Wash Dishes todo 23.03.2025 16:57:32  24.06.2025 18:59:22. time.Now().Format("02.01.2006 15:04:05")


    id: Уникальный идентификатор для задачи
    Описание: Краткое описание задачи
    статус: состояние задачи (todo, inde-degress, done)
    Создано в: Дата и время, когда была создана задача
    обновлено в: дата и время, когда задача была обновлена в последний раз

Обязательно добавьте эти свойства в файл JSON при добавлении новой задачи и обновите их при обновлении задачи.
